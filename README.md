# Portfolio - Yuri

Portfólio pessoal desenvolvido para apresentar minha trajetória como 
desenvolvedor full stack, meus projetos e tecnologias estudadas.

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (variáveis customizadas, Flexbox, mobile-first)
- JavaScript (vanilla)
- [Prism.js](https://prismjs.com) — syntax highlighting
- [Lucide Icons](https://lucide.dev) — ícones SVG

## ✨ Funcionalidades

- Tema claro/escuro com persistência
- Menu de navegação responsivo (drawer mobile)
- Seção de projetos com hover interativo
- Layout mobile-first totalmente responsivo

## 🚀 Como rodar localmente

\```bash
git clone https://github.com/Yurik11-dev/Portfolio.git
\```

Abra o arquivo `index.html` com a extensão Live Server do VS Code, 
ou qualquer servidor local de sua preferência.
